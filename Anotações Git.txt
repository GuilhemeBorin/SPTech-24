
Consultasr usuário configurado:
git config --global user.name
git config --global user.email

Configurar novo usuário:
git config --global user.name 'seu-nome-de-usuario'
git config --global user.name 'seu-email@sptech.school'

Limpando usuário configurado:
git config --global --unset user.name
git config --global --unset user.email


*Caso ao executar um comando para consultar um usuário após limpar, significa que não há mais nenhum usuário conectado*
